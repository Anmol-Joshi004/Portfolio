Project 2

Face Swap

This project creates a pipeline to swap faces in a video, replacing the original person's face with another provided by the user. Using deep learning and face manipulation, it preserves expressions, movements, and lighting for a seamless result. Frames are extracted, faces detected and aligned, then a model like DeepFaceLab is used to overlay the new face. The frames are recombined into a video with FFmpeg. This approach enables realistic face swaps for applications in entertainment and content creation.
